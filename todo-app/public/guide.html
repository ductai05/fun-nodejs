How to Run Todo App

Follow these steps to get your full-stack todo app running

1

Open Terminal

Open terminal/command prompt in your project directory

cd /path/to/your/todo-app
2

Install Dependencies

Install required Node.js packages

npm install
3

Setup Database

Initialize MongoDB with sample data

npm run setup
4

Start Server

Launch the development server

npm run dev
5

Open Browser

Access your app in the browser

http://localhost:3000
⚠️ Important: Use localhost:3000, not Live Server (127.0.0.1:5500)

Troubleshooting

❌ "Cannot GET /" error

Make sure you're accessing localhost:3000, not using Live Server

❌ "Failed to fetch" error

The Node.js server is not running. Run npm run dev first

❌ "MongoDB connection error"

Check your .env file has correct MONGODB_URI

❌ "Port 3000 already in use"

Kill existing process: killall node or change PORT in .env
